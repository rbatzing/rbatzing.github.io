Tools of Choice

An inventory of software in current and active use

General programming

MinGW C is part of a Windows port of the GNU Compiler Collection includes front ends for several languages, namely C, C++, Objective-C, Fortran, Java, Ada, and Go. GCC was originally written as the compiler for the GNU operating system but has been ported to Mac, Linux, and Windows environments. As with all of the GNU system software, GCC was developed to be 100% free software.

Java has become a popular and important environment for building application used by over 9 million developers worldwide. With its object-oriented design, Java has become a global standard for developing and delivering mobile applications, games, Web-based content, and enterprise software. Because the compiler targets a standard Java virtual machine that has been implemented on a wide range of hardware, Java delivers compiled applications that can be run on numerous computing environments in a manner that is close to achieving the goal of "Compile once and use everywhere".

Python is an easy to learn, but powerful programming language that combines high-level data structures with a simple, yet effective approach to object-oriented programming. This make this language easy to learn and ideal for research and rapid development. Python's elegant syntax and dynamic typing, together with its interpreted nature and extensive collection of libraries for a wide range of problem domains, make it an ideal language for scripting and rapid application development in many areas on most platforms.

Ruby is a general-purpose programming language that is fully object-oriented. While the syntax was inspired by Perl it was heavily influenced by Smalltalk, Eiffel, Lisp and Java. It also has a dynamic data type system, automatic memory management and full support for various non-Roman encoding systems. Ruby was first developed in the mid-1990s by Yukihiro Matsumoto in Japan and was designed to support multiple programming paradigms, namely functional, object oriented, imperative and reflective. Ruby on Rails has become a killer application of Ruby by facilitating controller-model-view (CMV) web development with utilities to support by agile programming.

Specialized programming

Metafont is a programming language for implementing outline fonts. Glyph shapes are represented as a collection of filled regions and strokes drawn with fixed width pens following pre-defined pen paths. The system is capable of describing a wide-range of fonts allowing for the description of numerous calligraphic, serif and sans-serif fonts from a single mathematical description by using global attributes to define font specific characteristics of glyphs.

GNU Octave is a high-level interpreted language, primarily intended for numerical computations. It provides capabilities for the numerical solution of linear and nonlinear problems, and for performing other numerical experiments. It also provides extensive graphics capabilities for data visualization and manipulation. Octave is normally used through its interactive command line interface, but it can also be used to write non-interactive programs. The Octave language is quite similar to Matlab and can run many programs designed for that system.

PHP is an open source programming language designed for adding program control and structure to the development of webpages and in the delivery of web content. This is achieved by embedding PHP script commands into the basic structure of HTML pages. The recursive name PHP is derived from recursive descriptive name: "PHP Hypertext Preprocessor"

Processing is an open source programming language designed for creating images, animations, and graphical interactions. The language originated as an attempt to create a software sketchbook and to teach fundamentals of computer programming within a visual context. Processing has evolved into a tool for generating finished professional artistic work.

R is a programming language designed to facilitate statistical analysis and graphical representation of data. The R language is open-source and has been used globally for developing statistical software and for data analysis of datasets of a wide range of sizes and types. The typical R language session arises from the execution of either a series of command lines or a pre-programmed script. There is a growing number of graphical user interfaces that include plug-ins to popular spreadsheets like Microsoft Excel and Open Office Calc.

Scratch is an environment to enable new students of programming to rapidly get results in their attempts to learn computer programming without having to learn syntactically correct writing first. Created by the MIT Media Lab, it was designed to facilitate learning through playful experimentation and the creation of graphics, games and animation projects. The development interface allows users to step through the programmed commands in manner that resembles an animated flowchart.

SQLite  is a software library that implements a SQL database server as a self-contained application that implements a transactional SQL database engine without the need to bind server ports. SQLite is the most widely deployed SQL database engine in the world and requires no configuration.

XeLaTeX is a multi-lingual version of LaTeX that allows users to typeset documents captured in Unicode (UTF-8). Like pdfLaTeX, XeLaTeX is converts text into indexed PDF files. The key difference is that the system supports both the fonts and the language specific line break routines of the underlying operating system. It provides support for multiple non-Roman scriptsas well as graphics and color.

Graphics and media development

Audacity: is an open-source sound editor that provides a graphic interface for the manipulation of audio waveforms. This program supports a wide range of audio transformations and  audio file standards.

Blender is a 3-D rendering  system that  applies textures to wireframes. The result scenes can be further processed to ray tracing, and ambient lighting effects as well as camera angles and position. The rendering of individual frames for a movie can also be automated.

Lily Pond is a sheet music compiler that accepts ASCII input and generates high quality output one would expect from a music engraving program. It was designed to produce the high quality sheet music by bringing the aesthetics of traditionally engraved music to computer printouts. It is also provides a wide range of musical transformation.

OpenCV is an open library of C language functions developed to process images and to facilitate the development of computer vision applications.

OpenGL is an open library of C language functions developed to render 3-D images and to facilitate the development of realistic computer graphics.

Simple Directmedia Layer is an open library of C language functions developed to rapidly extract 2-D images and audio from video streams and to facilitate the development of video applications that can extract information and features in real time.
