General programming

MinGW C is part of a Windows port of the GNU Compiler Collection which includes front ends for several languages, namely C, C++, Objective-C, Fortran, Java, Ada, and Go. GCC was originally written as the compiler for the GNU operating system but has been ported to Mac, Linux, and Windows environments. As with all of the GNU system software, GCC was developed to be 100% free software, free in the sense that it respects the user's freedom. While the compiler works as a stand-alone utility, it has been encorporated into IDE like Eclipse to provide a programming environment exercise similar to that found with Visual Studio.

Java has become a popular and important environment for building application used by over 9 million developers worldwide. With its object-oriented design, Java has become a global standard for developing and delivering mobile applications, games, Web-based content, and enterprise software. Java delivers applications portability across disparate computing environments in a manner that is close to achieving the goal of "Compile once and use everywhere".

Python is an easy to learn, but powerful programming language that combines high-level data structures with a simple, yet effective approach to object-oriented programming. Python's elegant syntax and dynamic typing, together with its interpreted nature and extensive collection of libraries for a wide range of problem domains, make it an ideal language for scripting and rapid application development in many areas on most platforms.

Ruby is a general-purpose programming language that is fully object-oriented. While the syntax was inspired by Perl it was heavily influenced by Smalltalk, Eiffel, Lisp and Java. It also has a dynamic type system, automatic memory management and full support for various non-Roman encoding systems. Ruby was first developed in the mid-1990s by Yukihiro Matsumoto in Japan and was designed to support multiple programming paradigms, namely functional, object oriented, imperative and reflective. Ruby on Rails is a web development environment with utilities to support by agile programming.

Specialized programming

Metafont is a language for describing outline fonts. Glyph shapes primarily consist of a collections of filled regions and strokes drawn with fixed width pens following a defined pen path. The system is capable of describing a wide-range of fonts allowing for the description of numerous calligraphic, serif and sans-serif fonts from a single mathematical description by using global attributes to define specific font characteristics.

GNU Octave is a high-level interpreted language, primarily intended for numerical computations. It provides capabilities for the numerical solution of linear and nonlinear problems, and for performing other numerical experiments. It also provides extensive graphics capabilities for data visualization and manipulation. Octave is normally used through its interactive command line interface, but it can also be used to write non-interactive programs. The Octave language is quite similar to Matlab so that most programs are easily portable.

PHP is an open source programming language designed for adding program control and structure to the development of webpages and in the delivery of web content. This is achieved by embedding PHP script commands into the basic structure of HTML pages. The recursive name PHP is derived from name "PHP Hypertext Preprocessor"

Processing is an open source programming language designed for creating images, animations, and graphical interactions. The language originated as an attempt to create a software sketchbook and to teach fundamentals of computer programming within a visual context. Processing has evolved into a tool for generating finished professional work.

R is a programming language that was designed for the development of functions that facilitate statistical computing and graphical representation of data. The R language is open-source and has been used globally for developing statistical software and the analysis of data analysis. The typical R language session arises from the execution of either a series of command lines or a pre-programmed script. There is a growing number of graphical user interfaces that include plug-ins to popular spreadsheets like Microsoft Excel and Open Office Calc.

Scratch is an environment to enable young beginners to rapidly get results in their attempts to learn computer programming without having to learn syntactically correct writing first. Created by the MIT Media Lab, it was designed to facilitate learning through playful experimentation and the creation of graphics, games and animation projects. The development interface allows users to step through the programmed commands in manner that resembles an animated flowchart.

SQLite is a software library that implements a SQL database server as a self-contained application that implements a transactional SQL database engine without the need to bind server ports. SQLite is the most widely deployed SQL database engine in the world and requires no configuration. The source code for SQLite is in the public domain.

XeLaTeX is a multi-lingual version of LaTeX that allows users to typeset documents captured in Unicode (UTF-8). Like pdfLaTeX, XeLaTeX is converts text into indexed PDF files. The key difference is that the system supports both the fonts and the language specific line break routines of the underlying operating system. It provides support for multiple non-Roman scripts in the same document as well as graphics and color.
