Course Syllabus

CSCI A340 Introduction to Web Programming

Spring 2009- 2:30-3:45PM NS164

Instructor:

Robert Batzinger, rbatzing@iusb.edu, NS160C

Course Description:

Introduction to Web programming. This course will focus on the development of services which allow clients to interact with data and documents on a server. The course will use PHP to demonstrate client-server interaction and to explore ways to improve the quality of this interaction through various HTML, CSS, and JavaScript techniques. Throughout the course, assignments and projects will provide practical hands-on experience which culminate in building a practical Web site with dynamic elements that use both client-side and server-side techniques. Prerequisite: A201 or C101.

Programming environment

  • PHP5 and SQLite components of XAMPPlite available from http://www.xampp.org
  • It is intended that this software would be downloaded onto a flash drive that could be used on campus as well as at home.
  • The class project this year will be to build a web site for submitting announcements for display. Exercises throughout  the term will be aimed at building various components of  the basic website. Student will undertake a project to enhance specific features or adding new services to this website.

General Course Outline:

  • Getting Started: Overview of the course; Installing and testing XAMPPlite; Running PHP on the server, via HTTP, embedded in webpages
  • The nature of http communications: Server-side activity, client-side activity, the http protocol
  • PHP essentials: Variables; Strings, Arrays and Hashes; Regular expressions; Program Control; Functions and modules
  • Client – Server interaction: Responding to simple requests; Enhancing the quality of the display; Requesting information from the client; Responding to inputs from the client
  • Object-oriented techniques: Modeling the data objects in a website; Class definitions in PHP; Using objects in PHP; Interacting with clients using objects
  • Interacting with databases: Introduction to SQL; Building and interacting with a SQLite database; Interacting with a database with PHP; Building a versatile function library
  • Building a website: Modeling the states of a website; Identifying the major components required; Establishing standards; Session variables; Building a functional prototype; Authentication and role-based access to services; Security issues; Data verification
  • Common web services: A Guestbook; Shopping cart; User administration; Content management; Sending email; Uploading and downloading files
  • Presentations: Demonstration of web projects; Current topics and future directions
