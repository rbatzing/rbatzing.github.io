The RSS makes it possible to deliver links to over a dozen news items and the related film clips, audio bites and news stories.

The following Ruby program does the following:

1) Connect to the internet
2) Download the contents of the current New York Times USA national news.
3) Store the contents of the file in the string buffer.
4) Parse the string into XML components.
5) Extract each news item separately
6) Display the title of a news item
7) Display the description of the news item


require 'net/http'
require 'rexml/document'

url = 'http://feeds.nytimes.com/nyt/rss/US'

# STEPS 1,2,3
xml_data = Net::HTTP.get_response(URI.parse(url)).body

doc = REXML::Document.new(xml_data)
 
doc.elements.each('rss/channel/item') do |item|
    puts "\n\n"
    puts item.elements['title']
   
    txt =  item.elements['description'].text
    puts txt.sub(/\<br.+/m,'')

    puts item.elements['pubDate'].text
end
