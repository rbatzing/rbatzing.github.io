Extending petminder.rb

The file petminder.rb contains the Ruby code for an application to monitor the feeding and watering of pets.
 
For this assignment, you are being asked to extend this program in the following way:

1) Add a new subclass of Pet called Rock. This class of pets eats 'nothing' and uses 'nothing' as a water container. It never needs to be watered or fed and is always 'content'. Users should be able to create and monitor instances of Rock Pets in the same way they can with pets that are Horses, Dogs or Cats. Users should be allowed to attempt to feed and water their pet Rock even though these actions will only change the time of action and the Rock status will remain the same. You will need to provide new  local definitions for the predicate methods of Pet and make appropriate changes to methods in other classes to access instances of this new class.

2. Add a new method to Collection called feed_and_water. The purpose of this method is to feed all animals requiring food and provide water to all animals requiring water. The method should not attempt to feed animals that have already be fed or water animals that have already be watered. You will need to make appropriate changes to methods in other classes to allow the user to access this new method.

Please make all appropriate changes to your copy of petminder.rb and test it using Ruby. Once you are satisfied that it is working, please put a copy of the program and your output demonstrating that your additions work in your dropbox.
